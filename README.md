# kafka-monitoring-lab
Kafka monitoring lab - Prometheus, Grafana, and Kafka Exporter via Docker Compose. 
# Kafka Monitoring Lab

## Setup

Open in GitHub Codespaces, then run:

docker-compose up -d

| Service    | Port | Access                |
|------------|------|-----------------------|
| Grafana    | 3000 | Codespaces Ports tab  |
| Prometheus | 9090 | Codespaces Ports tab  |

Follow the lab document for step-by-step instructions.
